# BINARY ASCII

ULTIMATE VERSATILE KEYMAP
You don't need to setup unicode config.    
Only ascii code.

```
,---------------.
|   0   |   1   |
|-------+-------|
| ENTER | CLR/BS|
`---------------'
```

1. Input 8bit ascii code with 0,1 keys.
2. Push Enter

### CLR/BS
- If internal buffer is 0, clear buffer.
- If not, Backspace
