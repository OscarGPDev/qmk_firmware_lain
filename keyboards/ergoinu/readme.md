ErgoInu
===

[ergoinu](https://i.imgur.com/4CCM8Vl.jpg)

A (Not Portable But Small) Ergonomic Split Keyboard.

Keyboard Maintainer: [hsgw](https://github.com/hsgw/)   [@hsgw](https://twitter.com/hsgw)
